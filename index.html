Projects

During my career I have contributed to various projects in different disciplines.Here are some examples:


  • File and Data Structure
  • Operating Systems
  • Computer Graphics
  • Computer Gaming

File and Data Structure project was written in Java under windows and Ubuntu Linux operating systems.The goal was to create a system like MySQL. During this project the class had 1 semester to implement this system, and were given the project in nine parts to complete with design specifications and a library of keywords to implement.


Operating Systems was an interesting class.The goal was to create a simple operating system that could create and destroy processes, under this system the group had to write in the C programming language using the Turbo C compiler.This was an interesting project and consisted of many long nights supplimented with Redbull.


Computer Graphics was various projects during the course of a semester.Most of these projects were written using OpenGL in windows to get the basics.The final project was a big scene with multiple light sorces and camera manipulation.


Computer Gaming was taken in my last semester at WVU. Multiple simple projects with a term project were programmed.Various technologies were used including PyGame,processing,and some music software.The term project was sponsored by SleepWell a WVU program to help students with good sleeping habits.
